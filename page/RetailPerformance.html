Retail Performance Record

Description

The retail performance record system is designed to record the data that comprise business transitions for the sale of units of each brand to the retailers. By using this application the organization can able to manage the transition by retail outlet name or by date period and then can review the purchase history of the retailers.

Features

  • Brand
    • ü This feature is to store brand data.
  • Outlet
    • ü This feature is to store the information of each outlet retail.
  • Outlet Records
    • ü This feature is to record the purchase history of each outlet retail on each brand.
  • Export Data
    • ü The ability of the application to be able to produce the custom purchase history.
    ü The data can be exported by date period and retail outlet name.
  • Import Brand
    • ü This feature is to import brand data using CSV file format.
  • Import/Export Outlet
    • ü This feature is to import or export outlet data using CSV file format.

Summary

This application is aim to assist the organization to handle the outlet retail transitions. By using these transition records you can analyze the detail of each purchase operation.
